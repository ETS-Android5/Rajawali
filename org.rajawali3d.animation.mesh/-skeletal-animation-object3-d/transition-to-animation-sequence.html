transitionToAnimationSequence

open fun transitionToAnimationSequence(sequence: SkeletalAnimationSequence!, duration: Int): Unit

Transition to a new SkeletalAnimationSequence with the specified duration in milliseconds. This method will use a LinearInterpolator for interpolation. To use a different type of interpolator use the method that takes three arguments.

Parameters

sequence - SkeletalAnimationSequence!: The new SkeletalAnimationSequence to transition to.

duration - Int: The transition duration in milliseconds.

open fun transitionToAnimationSequence(sequence: SkeletalAnimationSequence!, duration: Int, interpolator: Interpolator!): Unit

Transition to a new SkeletalAnimationSequence with the specified duration in milliseconds. The Interpolator is an Android SDK Interpolator and can be one of AccelerateDecelerateInterpolator, AccelerateInterpolator, AnticipateInterpolator, AnticipateOvershootInterpolator, BounceInterpolator, CycleInterpolator, DecelerateInterpolator, LinearInterpolator, OvershootInterpolator.

Parameters

sequence - SkeletalAnimationSequence!: The new SkeletalAnimationSequence to transition to.

duration - Int: The transition duration in milliseconds.

interpolator - Interpolator!: The Interpolator
