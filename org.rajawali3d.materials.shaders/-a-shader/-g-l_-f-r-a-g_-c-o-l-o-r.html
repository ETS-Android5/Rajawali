GL_FRAG_COLOR

protected val GL_FRAG_COLOR: AShaderBase.GLFragColor!
protected val GL_FRAG_COLOR: AShaderBase.GLFragColor!

Defines the color of the current fragment. This is used in the fragment shader to write the final fragment color to. This corresponds to the gl_FragColor GLSL variable.
