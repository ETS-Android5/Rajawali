addUniform

protected open fun addUniform(var: AShaderBase.IGlobalShaderVar!): AShaderBase.ShaderVar!

Add a uniform. The uniform qualifier is used to declare global variables whose values are the same across the entire primitive being processed. All uniform variables are read-only and are initialized externally either at link time or through the API. The link time initial value is either the value of the variable's initializer, if present, or 0 if no initializer is present. Sampler types cannot have initializers.

Parameters

var - AShaderBase.IGlobalShaderVar!: A global shader variable.
