setRotation

open fun setRotation(quat: Quaternion!): ATransformable3D!

Sets the rotation of this ATransformable3D by the rotation described by the provided Quaternion. If this is part of a scene graph, the graph will be notified of the change.

Parameters

quat - Quaternion!: Quaternion describing the additional rotation.

Return
